Stock Stalker — AI Signal Engine
A Bloomberg-style dark terminal stock screening dashboard powered by Claude AI and real-time Yahoo Finance data.
Features

Real-time multi-timeframe signal scanning (M5, M15, M30, H1, D1)
Claude AI automatically ranks and selects top 5 bullish or bearish prospects
120+ stocks across 8 preset watchlists — Mega Cap, Tech, Finance, Health, Energy, Sector ETFs, Dow Jones 30, and All
Dual scoring system — bearish score (red dots) and bullish score (green dots) computed separately
Relative Volume (RVOL) detection with blue dot volume confirmation
One-click filters: Top Picks, 4+ Red, Volume, Bearish, Bullish
Auto-refresh every 5 minutes for live market sessions
Demo mode with instant simulated data when markets are closed
Click any stock for AI-generated trade rationale

How to use

Select a watchlist preset or type your own symbols
Press ⚡ LIVE ANALYSIS during market hours (9:45 AM – 4:00 PM ET)
Switch between BEARISH and BULLISH filters — AI re-ranks picks automatically
Best time to run: 9:45–10:15 AM ET for highest-probability signals

Tech stack: Vanilla HTML/CSS/JS · Anthropic Claude API · Yahoo Finance API
